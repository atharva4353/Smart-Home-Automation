🎯 Objective

This project demonstrates how a Raspberry Pi can control household appliances using hand gestures detected through a USB camera.

A 4-channel relay module is used to control an LED, fan, buzzer and water pump.

🛠 Required Components

  • 🍓 Raspberry Pi
  • 📷 USB Camera
  • 🔌 4-Channel Relay Module
  • 〰️ Jumper Wires
  • 🔋 Power Supply for Raspberry Pi
  • ⚡ External Power Supply for Appliances
  • 💡 LED
  • 🌀 Fan
  • 🔊 Buzzer
  • 💧 Water Pump

📋 Procedure

  1. Download the project files from this webpage.
  2. Open the Terminal on the Raspberry Pi.
  3. Create a Python virtual environment.
  4. Activate the virtual environment.
  5. Install all required Python libraries.
  6. Connect the USB camera to the Raspberry Pi.
  7. Connect the 4-channel relay module according to the connection table.
  8. Create or download the Python file named Home_Automation.py.
  9. Run the Python program.
  10. The camera starts and detects the user's hand.
  11. Show the mapped finger to control the corresponding appliance.
  12. The selected appliance switches ON or OFF through its relay channel.

⚠ Troubleshooting

Problem Solution
Camera not detected Reconnect the USB camera and restart the Raspberry Pi.
Relay not switching Check VCC, GND, GPIO wiring and relay active-LOW logic.
Module not found Activate the virtual environment and install the required libraries.
Permission denied Run the program with the correct permissions and verify GPIO access.
Wrong appliance activates Verify the GPIO numbers and finger mapping table.
